Power Apps Send to Claude

Send a message to Claude

For when you're on the laptop — post a formula, error, or screenshot. Claude picks it up on the main PC.

You can paste a formula or error message here. Screenshots get attached on the GitHub page that opens next.

GitHub issue page opened in a new tab. Attach any screenshots by dragging them onto the body field there, then click Submit new issue.

How it works

1Fill in the subject and message above.
2Click Open GitHub to send — a new tab opens with the issue pre-filled.
3Drag any screenshots onto the issue body, then click Submit new issue.
4Claude picks it up on the main PC and actions it.
Screenshots: on the GitHub page that opens, just drag a screenshot image directly onto the text box — GitHub uploads it automatically and inserts a link. You can attach as many as you need.
